process.env

process.env 会返回包含用户环境的对象，来自 Node.js

参考链接：http://nodejs.cn/api/process/process_env.html

process.nev.VUE_APP_BASE_API

项目需要在不同环境下进行运行(开发，生产，测试等)，process.nev.VUE_APP_BASE_API可以避免多次切换请求的地址以及相关的配置。

当需要用到该变量时可以直接使用 process.nev.VUE_APP_BASE_API 进行取值
