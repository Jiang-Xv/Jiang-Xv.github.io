问题描述

使用 v-for 循环渲染后，在未改变 CSS 的情况下，直接写入的与通过 v-for 循环渲染的样式不同

出现原因

同一条数据的 title 与 note 分在了两个对象中，循环渲染时将每个 el-col 外都自动套上了一个 el-row，
因此两个 el-col 分在了两行中

解决方案

将同一条数据的 title 与 data 写在一个对象中
